Installation

There are 6 steps to installing this module.
1. Install the 'core' of the portal first.
2. Unzip the contents into the root of your site.
3. Upload all the files and folders to your web folder
4. Run the createForums.asp file to create the db tables and insert the default data
5. Add the include file to fp_custom.asp in your site root as outlined below.
6. Delete the createForums.asp file from your server.

Open the fp_custom.asp file in your favorite editor.
Around line 20 you will see this...
<!--include file="modules/forums/fp_forums.asp"-->
change that line to this...
<!--#include file="modules/forums/fp_forums.asp"-->
Save and upload.
