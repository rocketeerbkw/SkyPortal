Installation

There are 6 steps to installing this module.
1. Install the 'core' of the portal first.
2. Unzip the contents into the root of your site.
3. Upload all the files and folders to your web folder
4. Run the createArticles.asp file to create the db tables and insert the default data
5. Add the include file to fp_custom.asp in your site root as outlined below.
6. Delete the createArticles.asp file from your server.

Open the fp_custom.asp file in your favorite editor.
Around line 22 you will see this...
<!--include file="modules/articles/fp_articles.asp"-->
change that line to this...
<!--#include file="modules/articles/fp_articles.asp"-->
If it isn't there, then copy and paste the line above into your fp_custom.asp

Save and upload.

Notes

You can add the items to your front page through your admin >> managers >> layout manager
The Modules/articles/article_custom.asp file controls what appears as the "side items" in the article pages.
