SkyPortal Forms Module

Give site administrators the ability to generate and customize forms online and in real time.

Info
Version: 2.5
Author: Brandon Williams (Battousai)
Copyright: See included license file
Compatibility
SkyPortal: RC7 - v1.0
Database: Access (tested)
MSSQL (not tested)
Dependency: SkyPortal Core

Installation

New Install

  1. Upload all files in the zip to root of your SkyPortal installation
  2. Add <!--#include file="modules/forms/fp_forms.asp" --> to fp_custom.asp (HINT: next to all the other ones)
  3. Run createForms.asp
  4. Delete createForms.asp
  5. Enjoy :)

Upgrade

  1. Upload all files in the zip to root of your SkyPortal installation, overwriting the previous version
  2. Run createForms.asp
  3. Delete createForms.asp
  4. Enjoy :)
