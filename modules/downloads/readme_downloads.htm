New Installation

There are 6 steps to installing this module.
1. Install the 'core' of the portal first.
2. Unzip the contents into the root of your site.
3. Upload all the files and folders to your web folder
4. Add the include file to fp_custom.asp in your site root as outlined below.
5. Run the createDownloads.asp file to create the db tables and insert the default data
6. Delete the createDownloads.asp file from your server.

Open the fp_custom.asp file in your favorite editor.
Around line 21 you will see this...
<!--include file="modules/downloads/fp_dl.asp"-->
change that line to this... or add this line if it isnt already there
<!--#include file="modules/downloads/fp_dl.asp.asp"-->
Save and upload.

Upgrading existing installation

There are 6 steps to Upgrading this module.
1. BACK UP YOUR CURRENT DATABASE FIRST!
2. BACK UP YOUR FILES/DOWNLOADS/ FOLDER!
3. Unzip the contents into the root of your site.
4. Upload all the files and folders to your web folder.
5. Run the createDownloads.asp file to upgrade your existing installation.
6. Delete the createDownloads.asp file from your server.

Notes

You can add the items to your front page through your admin >> managers >> layout manager
The Modules/downloads/fp_dl.asp file controls what appears as the "side items" in the download module pages.
